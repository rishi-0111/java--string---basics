# Java String Practice

This repository contains beginner-level Java programs for practicing **String concepts**.
These problems help in understanding common **string operations used in coding interviews and problem solving**.

---

## 📚 Topics Covered

* String manipulation
* Character operations
* Loops and conditions
* Arrays and String arrays
* Basic algorithmic thinking

---

## 🧠 Programs Included

1. Reverse String
2. Count Vowels in String
3. Check Palindrome String
4. Remove Spaces from String
5. Print Characters One by One
6. Count Characters in String
7. Reverse Each Word in a Sentence
8. Count Words in a Sentence
9. Find Duplicate Characters
10. First Non-Repeating Character
11. Check Anagram Strings
12. Character Frequency
13. Remove Special Characters

---

## 🛠 Concepts Used

These Java methods and concepts are used throughout the programs:

* `length()`
* `charAt()`
* `toCharArray()`
* `split()`
* `replace()`
* `StringBuilder.reverse()`
* `Arrays.sort()`
* `Arrays.equals()`
* `Character.isLetterOrDigit()`
